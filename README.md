# Timecode Calculator for Unity
SMPTE Timecode calculation utility.

## Frame rate options

| Frame Rate | Drop Frame | 
| ---------- | ---------- | 
| 30         | False      | 
| 29.97      | True       | 
| 60         | False      | 
| 59.97      | True       |
